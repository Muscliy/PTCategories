Useful categories for Foundation and UIKit

Installation
===============

### CocoaPods
1、Add `pod 'PTCategories'` to your project Podfile.
2、Run `pod install` or `pod update`.

Requirements
==============
This library requires `iOS 8.0+` and `Xcode 7.0+`.
